Công cụ cài đặt Việt hóa hỗ trợ phiên bản DF 50.10 trở lên, bao gồm các phiên bản 51, 52 và 53.
Steam, itch.io, phiên bản cổ điển cho cả nền tảng Windows và Linux.

Hướng dẫn ngắn gọn về cách cài đặt Việt hóa:

- Tải xuống công cụ, giải nén, và chạy file  `dfint-installer`.
- ("Mở") công cụ và tìm đến đường dẫn cài đặt game rồi chọn file  (`Dwarf Fortress.exe` hoặc `dwarfort`). Hoặc bạn có thể đặt file  `dfint-installer` vào thư mục cài game, nó sẽ tự động tìm file Dwarf Fortress.exe.
- Chọn ngôn ngữ Vietnamse hoặc ngôn ngữ cần dịch, sau đó nhấn "Cập nhật".
- Vào game và thưởng thức
- Để cập nhật bản dịch hoặc tùy chỉnh phiên bản mới hơn của trò chơi, hãy chạy lại trình cài đặt (bảo đảm trò chơi đã tắt) rồi nhấn "Cập nhật".

Nếu bạn gặp sự cố khi sử dụng (ví dụ: bạn đang sử dụng Windows 7 hoặc 8), bạn có thể sử dụng [package-builder](https://dfint-package-build.streamlit.app){:target="_blank"} để thay thế.
