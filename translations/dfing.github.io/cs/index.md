Instalátor lokalizace podporuje verze DF 50.10 a novější, včetně verzí 51, 52 a 53.
Jsou podporovány verze ze Steamu, itch.io i klasické verze pro Windows i Linux.

 Stručné pokyny k instalaci překladu:

-  Stáhněte instalátor, rozbalte jej a spusťte soubor ` dfint-installer` .
-  Vyberte („Otevřít“) spustitelný soubor hry (`Dwarf Fortress.exe` nebo `dwarfort`).
Případně můžete soubor `dfint-installer` umístit do adresáře hry — v tom případě automaticky najde spustitelný soubor hry sám.
-  Zvolte jazyk překladu a poté stiskněte ‚Update‘ (Aktualizovat).
- Spustit hru.
- To update translation or configuration for a newer version of the game, run the installer again (while the game is shut down) then press "Update".

If you have some troubles with the installer (e.g. you are useing Windows 7 or 8), you can use the [package-builder](https://dfint-package-build.streamlit.app){:target="_blank"} instead.
